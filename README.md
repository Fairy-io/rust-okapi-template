## Prerequisites

[Install rust and cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)

Run following command to enable watch mode:
```sh
cargo install cargo-watch
```

##

In order to start application in watch mode:
```sh
cargo-watch -x run
```

To run tests:
```sh
cargo-watch -x test
```
